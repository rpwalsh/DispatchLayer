# Walsh Predictive Core

Deterministic evidence-weighted reasoning engine for renewable energy operations intelligence.

## Modules

- **signal_state**: Signal normalization and range validation
- **evidence_graph**: Weighted evidence chain construction
- **confidence**: Evidence-weighted confidence scoring with recency decay
- **residuals**: Production residual computation
- **reconciliation**: Forecast bias correction and reconciliation
- **causal_attribution**: Causal hypothesis ranking for wind/solar underproduction
- **forecast_bounds**: p10/p50/p90 bounds from historical error distributions
- **decision_trace**: Auditable step-by-step reasoning traces
